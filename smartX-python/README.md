# SmartX-cloud-python-web
SmartX-cloud-python-web


# 环境说明
windows用户，请执行
`SmartX-cloud-python-web\resource\script\windows`下的
`install-python-dev.bat`文件
启动方式：
	操作`命令行`进入该目录，输入ins点下tab键回车等待即可

# 目录说明
```java
	|---demo.py        指定本地图片人脸识别
	|---index.py       启动flask测试
	|---netPicDemo.py  网络图片人脸识别
```

# 启动说明
在本目录，执行如下命令
```
python index.py
访问localhost:5000 即可
```
